Lighweight Image Lazyload Script

A simple Lighweight script that lazy load your website images.

IT will help to improve your website performance and decrease HTTP Requests during the First-Page Load

Installation

  1. Add minified version of lazyload.js at the End of your HTML
  2. Images: Insert a placeholder to the src of your images
  3. Images: Put the real src to the data-src attribute
  4. Background Images: Put the background URL to the data-bg attribute
